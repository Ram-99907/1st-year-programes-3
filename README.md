# Student Management System (C)
#Description

This is a simple C program that accepts student details and displays them.

## Details Collected
- Roll Number
- Name
- Marks

## Purpose
This project is created to understand basic input and output operations in the C programing language.

## Concepts Used
- Variables
- printf() and scanf()
- Basic C program structure

## How to Run
1. Compile the program:
   gcc main.c
2. Run the executable file.

## Sample Output
Enter Roll Number: 101  
Enter Name: Karthik  
Enter Marks: 85.5
